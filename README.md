#  Controle de Gastos

##  Problema

Muitas pessoas têm dificuldade em controlar seus gastos diários.

# Solução

Uma aplicação simples em linha de comando para registrar e acompanhar gastos.

# Público-alvo

Pessoas que querem organizar suas finanças pessoais.

# Funcionalidades

* Adicionar gastos
* Listar gastos
* Calcular total gasto

# Tecnologias

* Python
* Pytest

# Como executar

```bash
python src/app.py
```

# Testes

```bash
pytest
```
